Blog create 2022-06-12

notes on how this blog is create. 介绍 gitee 和 github都提供静态网站托管服务。在gitee page的页面可以看到描述： Gitee Pages 服务 一个支持Jekyll、Hugo、Hexo静态网站的服务 使用帮助 https://gitee.com/help/articles/4136#article-header4 Jekyll、Hugo、Hexo 究竟是什么？ Jekyll、Hugo、Hexo 是简单的博客形态的静态站点生产机器。它有一个模版目录，其中包含原始文本格式的文档，通过 Markdown 以及 Liquid 转化成一个完整的可发布的静态网站，你可以发布在任何你喜爱的服务器上。Jekyll、Hugo、Hexo 也可以运行在 Gitee Pages 上，也就是说，你可以使用 Gitee 的服务来搭建你的仓库页面、博客或者网站，而且是完全免费的。 Jekyll 使用文档：https://www.jekyll.com.cn/docs/home/ Hugo 使用文档：https://gohugo.io/documentation/ Hexo 使用文档：https://hexo.io/docs/ markdown基本语法 https://markdown.com.cn/basic-syntax/ 技术选型 Jekyll基于ruby； hugo基于go; hexo基于nodejs； 选择hugo. 过程 参考：https://www.gohugo.org 在https://themes.gohugo.io/themes 选一个主题， git submodule add https://github.com/queensferryme/hugo-theme-texify.git themes/hugo-theme-texify 就可以clone下来。 当前选用了这个主题： https://themes.gohugo.io/themes/hugo-theme-texify/ 基本命令： $ hugo new site /path/to/site $ hugo new about.
